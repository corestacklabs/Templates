
# Provision_Database_AWS_Terraform

Provides sample configuration for creating a mysql or postgres instance.

### Prerequisites

AWS Account 
Microsoft Azure Subscription

## Deployment

Use terraform plan/apply to deploy the infra. Alternatively, click below to deploy the template directly to the corestack product
